# Welcome to My Static Site

This is the **home page** of my website, built with a Ruby static site generator!

## About This Site

This site is generated from Markdown files. You can edit this file (`content/index.md`) to change the home page content.

## Getting Started

- Add more Markdown files to the `content/` directory
- Customize the template in `templates/default.html` to change the design
- Run the generator to build your site
